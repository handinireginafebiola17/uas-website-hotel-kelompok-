## Deskripsi Project
Project ini merupakan website berbasis PHP yang dibuat untuk memenuhi
tugas Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web.

Website ini dikembangkan menggunakan HTML, CSS, JavaScript, PHP,
serta database MySQL yang dijalankan melalui XAMPP.
Project ini dikerjakan secara berkelompok sebagai bentuk penerapan
materi frontend dan backend yang telah dipelajari selama perkuliahan.

Fitur utama pada website ini meliputi:
- Pengelolaan data melalui form
- Dashboard admin
- Koneksi database
- Tampilan antarmuka website

## Catatan
Project ini dijalankan secara lokal menggunakan XAMPP
karena menggunakan PHP dan database MySQL.
Source code diupload ke GitHub untuk keperluan penilaian UAS.
